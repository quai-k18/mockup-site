# La Fenêtre — Mockup site

Quatre propositions visuelles pour le site web de **La Fenêtre**, glacier artisanal au Port de Morgat, Crozon.

Chaque fichier est une page HTML autonome (CSS inline, aucune dépendance).

## Styles

| Fichier | Direction |
|---|---|
| `index-style1-v2.html` | Éditorial chaleureux — typographie serif classique |
| `index-style2-v2.html` | Carte postale vintage estivale |
| `index-style3-v2.html` | Moderne et audacieux — accents néon |
| `index-style4.html` | Minimalisme luxe — épuré, grand blanc |

## Démo

Le site de démonstration est accessible via GitHub Pages. La page d'accueil liste les quatre styles et permet de les parcourir directement.

## Développement

Aucune étape de build. Ouvrir directement les fichiers `.html` dans un navigateur.
